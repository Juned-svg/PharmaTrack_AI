PharmaTrack AI is a production-ready data engineering and machine learning system designed to handle the high-velocity, unstructured telemetry data generated by laboratory instruments (e.g., Chromatography, Spectrometers).

The system ensures Data Integrity through strict schema enforcement and enables Predictive Maintenance by isolating mechanical anomalies using unsupervised learning.

🛠️ System Architecture
The pipeline operates in three distinct stages:

Ingestion & Tokenization: Uses a high-performance Regex engine to parse messy, multi-variate log files into structured Python objects.

Constraint Enforcement (Validation): Leverages Pydantic v2 to enforce runtime domain boundaries (e.g., Temperature/Pressure limits). Out-of-bounds records are automatically routed to a Quarantine Storage for audit trails, ensuring the main ML pipeline remains clean.

Anomaly Detection (AI): Fits a Scikit-Learn Isolation Forest model on validated data to detect structural outliers and erratic machine behavior without the need for human-labeled datasets.
